# Дурак с Машей

Мини-прототип Telegram Web App: серия партий в подкидного дурака один на один с Машей от первого лица.

## Запуск локально

Откройте `index.html` в браузере или поднимите любой статический сервер в этой папке.

```bash
python3 -m http.server 4173
```

После этого страница будет доступна по адресу `http://localhost:4173`.

## Для Telegram

Файлы уже подключают `https://telegram.org/js/telegram-web-app.js` и вызывают `Telegram.WebApp.ready()` / `expand()`. Для реального мини-приложения нужно разместить эту папку на HTTPS-хостинге и указать URL в настройках бота через BotFather.

## Что уже есть

- колода на 36 карт;
- козырь, добор до 6 карт, отбой и взятие;
- атака, защита и подкидывание по достоинству;
- Маша в интерфейсе и полноэкранные сцены после побед;
- 3 сердца игрока;
- 6 этапов прогресса Маши;
- сохранение прогресса по Telegram user id через `localStorage`;
- 100+ реплик Маши после ходов, отбоя, взятия и результата партии, с тоном по уровню;
- подкидывание карт, когда Маша берет, без преждевременного добора из колоды;
- повышение сложности от партии к партии: от сброса мелких карт до подсчета видимых карт без чтения руки игрока;
- мобильный интерфейс под Telegram.
